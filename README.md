# Recruit Open Platform

> Front-end version

## Build Guide

### In Develop Mode
```
npm i vue-cli -g
npm i
npm run serve
```
### Build For Production
```
npm run build
```

## Contribute Guide
Add `dev.rop.zjuqsc.com` or name like `*.zjuqsc.com` in your local `hosts`, and listening at `80` to make sure you have access to cookies belong to *QSCPassport*.
